# coordination-Post-app
卒業制作 - コーディネート投稿アプリ

## サービス概要
このアプリは、コーディネートを投稿し自分を表現できるだけではなく、フォロワーや友達、身近な人と交流し、オフ会やイベント前に自分を知ってもらうことで安心感を提示し、友達の場合はニュアンスコーデなど事前にファッションを合わせたりすることができるアプリです。服装やスタイルは、その人の内面、個性、価値観、心理状態、ライフスタイルなどを映し出す鏡のようなものであり、ネットで知り合い会ったことがない方のコーディネートを事前に知ることで、その人をイメージでき、安心につながります。また、自分のコーディネートの評価を受けることで、この服装はTPOに合っているか、他人とズレていないかを確認することができます。

## このサービスへの思い・作りたい理由
RUNTEQを通じて知り合った方と交流する機会がありました。初めてのこともあり、文章のやり取りだけで会ったことがない、どんな人かわからないまま電車と新幹線に乗って会うまでの間、体調が悪くなるほど緊張していました。しかし、実際に会ってみると変な人など1人もおらず、むしろ笑顔で話しかけてくれるような楽しい方がたくさんおり、あの緊張はなんだったのかと拍子抜けしたのと同時に安堵しました。事前にもう少し交流する人を知る術があれば、という思いでこのサービスを思いつきました。
私は外にいる時、歩いている人の服装を目で追ってしまうほどファッションが好きです。理由は前述の通り、その人の個性を映し出す鏡のようなものだからです。小さい頃の習い事で新体操やエアロビクスをしていた時、常に自分を鏡で見る環境にいました。社会人になった今はもうそういう機会はありませんが、振り返ると「自分を表現すること」が好きだったんだなと実感します。ダンスで自分を表現できる機会がなくなった今、ファッションは私にとって自分を表現する材料となっています。これが大好きな「ファッション」をアプリに取り入れたい理由です。

## ユーザー層について
10代〜30代
すでにコーディネートを写真に撮って投稿する習慣がある人に寄せた方が成功しやすいと思いました。

## サービスの利用イメージ
ユーザーはSNSのようにスマホで気軽に投稿できます。
オフ会に行く足が動かない方、どんな方がいるのか不安でオフ会を諦めてしまう方に利用してもらうことで、会う方のイメージをより想像でき、足を動かしやすくなります。
また、自分のコーディネートを投稿し評価してもらうことで、TPOに合っているか、他人とズレていないかという自分の不安も解消できます。

## ユーザーの獲得について
RUNTEQではオフ会が盛んなため、Xでオフ会の際の使用を促すことでサービスを届けたいと考えています。

## サービスの差別化ポイント・推しポイント
競合アプリ調査
アプリA（コーディネート検索アプリ）
・良い点：検索項目が多く、ユーザー層が幅広い
・悪い点：モデルの体型などが細かく伝わらない、いいねやコメント機能はない、すべて他人

アプリB（ファッション通販アプリ）
・良い点：老若男女問わずさまざまなファッションアイテムを購入できる、通知でお得な情報をゲットできる
・悪い点：レビューのない商品もある、おすすめ表示がすべてPR商品である

自分のアプリの差別化ポイント
・身長体型や天気、季節で検索できる
・フォロワーや友達など身近な人のコーディネートを知ることができる
・いいねやコメント機能で自分のコーディネートの評価を受けられる
