### 本サービスの概要（700文字以内）
本アプリは、オフ会やイベントなど「人と実際に会う前」に感じる不安を、コーディネートや雰囲気の共有を通じて和らげることを目的としたコーディネート投稿アプリです。
イベント主催者がイベントページを作成し、招待URLを共有することで参加者が集まり、当日の服装の雰囲気やテイスト、持ち物の目印などを事前に共有できます。
参加者全員の投稿を前提とせず、閲覧のみでも利用できるため、投稿に不慣れな人でも参加しやすい設計としています。

また、イベント開催時以外でも利用できる「自由投稿（事前相談用）」を用意し、次に人と会う予定を想定したコーディネート相談が可能です。
自由投稿では写真投稿を必須とし、具体的な相談がしやすい設計としています。

一般的な「いいね」機能は採用せず、投稿内容に応じたリアクションタグを選択する形式とすることで、数値評価による比較を避け、投稿者が「どう受け取られたか」を知れるようにしています。
本アプリは自己表現を競うSNSではなく、人と会う前の準備・相談に特化したサービスです。
