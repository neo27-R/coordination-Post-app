# coordination-Post-app
卒業制作 - コーディネート投稿アプリ

## サービス概要
本サービスは、オフ会やイベントなど「人と実際に会う前」に、イベントページを通じてコーディネートや雰囲気に関する情報を共有し、
参加者の心理的な不安を和らげることを目的としたコーディネート投稿アプリです。

主催者がイベントページを作成し、招待URLを共有することで参加者を巻き込める設計としています。
参加者全員の投稿を前提とせず、一部の参加者や主催者の投稿、または簡単な情報共有だけでも
当日の雰囲気を想像できるようにすることで、参加への心理的ハードルを下げます。

本サービスは「自分を魅せるためのSNS」ではなく、人と会う前の安心材料を提供することに特化しています。

## このサービスへの思い・作りたい理由
RUNTEQのオフ会に参加した際、参加前は「どんな人がいるのか分からない」という不安がありました。
しかし実際に参加してみると、気さくで話しやすい方ばかりで、事前の緊張が拍子抜けするほど安心した経験があります。
この体験から、「事前に相手の雰囲気を少しでも知ることができれば、人と会うことへの不安を軽減できるのではないか」と考えました。

私は日常的にファッションから人の個性を感じ取ることが多く、ファッションはその人らしさを知るための有効な手段だと感じています。
この“ファッションを通じて相手を想像できる体験”を形にしたいと考え、本サービスを企画しました。

## ユーザー層について
・10代〜30代
・コーディネートを写真に撮って投稿する習慣がある人
・オフ会やイベントへの参加に興味はあるが、不安を感じやすい人
・非対面での交流が中心で、実際に会うことにハードルを感じている人

既存のSNSと近い操作体験を提供することで、初めて利用するユーザーでも迷わず投稿できる設計を想定しています。

## サービスの利用イメージ
・主催者がオフ会・イベントページを作成
・招待URLを共有し、参加者がイベントに参加
・主催者や一部参加者がコーディネートや当日の雰囲気を投稿
・投稿は任意（閲覧のみの参加も可能）
・投稿が少ない場合でも、イベント情報や簡単な共有内容から雰囲気を想像できる
・いいねやコメントを通じて、TPOに合っているかを確認し、不安を軽減

## ユーザーの獲得について
RUNTEQではオフ会が活発に行われているため、オフ会主催者がイベントページを作成し、参加者に招待URLを共有することで、
自然な形でサービス利用が広がる導線を想定しています。

XなどのSNS上で利用事例を共有することで、実際の利用シーンを想起してもらい、サービス認知を高めます。

## サービスの差別化ポイント・推しポイント
競合アプリ調査
アプリA（コーディネート検索アプリ）
・良い点：検索項目が多く、ユーザー層が幅広い
・課題点：モデルの体型や実際の雰囲気が伝わりにくく、交流機能が少ない

アプリB（ファッション通販アプリ）
・良い点：幅広い商品ラインナップ、通知による情報提供
・課題点：PR中心の表示になりやすく、個人の雰囲気が分かりにくい

自分のアプリの差別化ポイント
・イベントページ方式を採用し、主催者主導で参加者を巻き込める
・参加者全員の投稿を前提としない設計
・身長、体型、天気、季節などの条件で検索可能
・フォロワーや友人など、身近な人のコーディネートを確認できる
・いいね・コメント機能による双方向のコミュニケーション
・「人と会う前の不安を和らげる」という明確な利用シーンに特化

・投稿作成 / 編集 / 削除

・いいね機能
・コメント機能

・フォロー機能
・投稿の保存（ブックマーク）機能
・タグ検索機能
・イベント名タグ機能
・身長体型タグ機能
・ブランドタグ機能
・季節、天気、シーンタグ機能
