# coordination-Post-app
卒業制作 - コーディネート投稿アプリ

## サービス概要
本サービスは、オフ会やイベントなど「人と実際に会う前」にコーディネートを投稿・共有することで、相手の雰囲気を事前に知り、心理的な不安を和らげることを目的としたコーディネート投稿アプリです。
フォロワーや友人など身近な人のコーディネートを確認できるため、当日の服装の方向性を事前にすり合わせたり、場の雰囲気を想像しやすくすることで、参加への心理的ハードルを下げます。
本サービスは「自分を魅せるためのSNS」ではなく、「人と会う前の安心材料を提供すること」に特化しています。

## このサービスへの思い・作りたい理由
RUNTEQのオフ会に参加した際、参加前は「どんな人がいるのか分からない」という不安がありました。
しかし実際に参加してみると、気さくで話しやすい方ばかりで、事前の緊張が拍子抜けするほど安心した経験があります。
この体験から、「事前に相手の雰囲気を少しでも知ることができれば、人と会うことへの不安を軽減できるのではないか」と考えました。

私は日常的にファッションから人の個性を感じ取ることが多く、ファッションはその人らしさを知るための有効な手段だと感じています。
この“ファッションを通じて相手を想像できる体験”を形にしたいと考え、本サービスを企画しました。

## ユーザー層について
・10代〜30代
・コーディネートを写真に撮って投稿する習慣がある人
・オフ会やイベントへの参加に興味はあるが、不安を感じやすい人

既存のSNSと近い操作体験を提供することで、初めて利用するユーザーでも迷わず投稿できる設計を想定しています。

## サービスの利用イメージ
・ユーザーはスマートフォンから気軽にコーディネートを投稿
・オフ会やイベント前に、参加者のコーディネートを事前に確認
・当日の服装の方向性や場の雰囲気を把握でき、安心して参加できる
・投稿に対するいいねやコメントを通じて、TPOに合っているか、他者とのズレがないかを確認できる

## ユーザーの獲得について
RUNTEQではオフ会が活発に行われているため、XなどのSNS上でオフ会参加時の活用例を共有することで、
実際の利用シーンを想起してもらい、サービス認知を広げていく想定です。

## サービスの差別化ポイント・推しポイント
競合アプリ調査
アプリA（コーディネート検索アプリ）
・良い点：検索項目が多く、ユーザー層が幅広い
・悪い点：モデルの体型などが細かく伝わらない、いいねやコメント機能はない、すべて他人

アプリB（ファッション通販アプリ）
・良い点：老若男女問わずさまざまなファッションアイテムを購入できる、通知でお得な情報をゲットできる
・悪い点：レビューのない商品もある、おすすめ表示がすべてPR商品である

自分のアプリの差別化ポイント
・身長、体型、天気、季節などの条件で検索可能
・フォロワーや友人など、身近な人のコーディネートを確認できる
・いいね・コメント機能による双方向のコミュニケーション
・「人と会う前の不安を和らげる」という明確な利用シーンに特化
